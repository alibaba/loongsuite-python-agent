## Version 0.1.0

### Added

- Initial release of LoongSuite Baggage Span Processor
- Support for prefix matching to filter baggage keys
- Support for prefix stripping to remove prefixes from baggage keys before adding to span attributes
- Integration with LoongSuite Configurator via environment variables
